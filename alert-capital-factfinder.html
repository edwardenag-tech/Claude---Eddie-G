One-time setup required to enable email submission

This form sends directly to marcus@alertloans.com.au via EmailJS (free, no server needed). Complete these steps once:

  1. Sign up free at emailjs.com
  2. Add an Email Service (Gmail recommended) → copy the Service ID
  3. Create an Email Template → set To Email to marcus@alertloans.com.au, Subject to {{subject}}, and HTML Content to {{message}} → copy the Template ID
  4. Go to Account → API Keys → copy your Public Key
  5. Open this HTML file in a text editor and replace the three placeholders near the top of the <script> block: YOUR_PUBLIC_KEY, YOUR_SERVICE_ID, YOUR_TEMPLATE_ID
